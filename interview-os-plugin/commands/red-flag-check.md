# Red Flag Check Command

Run this when something in the interview process feels off.

## Usage

```
/red-flag-check "[Describe what happened]"
```

Examples:
- `/red-flag-check "Interviewer was 20 minutes late and didn't apologize"`
- `/red-flag-check "They said 'we're like a family here'"`
- `/red-flag-check "They've rescheduled my interview 3 times"`

## What This Does

Immediately invokes **redflag-navigator** skill to:

1. **Categorize the Flag**
   - Organizational Chaos
   - Culture Toxicity
   - Respect Issues
   - Process Problems

2. **Assess Severity**
   - üü¢ GREEN: Probably fine
   - üü° YELLOW: Worth noting
   - üü† ORANGE: Concerning
   - üî¥ RED: Strong exit signal

3. **Decode the Concern**
   - What they said/did (surface)
   - What it really means (hidden motivation)
   - What it signals about working there

4. **Provide Response Strategy**
   - In-the-moment power move response
   - Follow-up actions to take
   - Validation research to run

5. **Decision Guidance**
   - Should you proceed?
   - Proceed with caution?
   - Walk away?

## Quick Red Flag Reference

### üü¢ Normal Interview Things (Don't Worry)

- One interviewer asks basic questions
- 1 day delay in scheduling
- Standard "tell me about yourself"
- Asking about salary expectations
- Checking references

### üü° Yellow Flags (Note But Don't Panic)

- One reschedule
- Interviewer slightly late (5-10 min)
- Generic interview questions
- Slow response time
- "Fast-paced environment"

### üü† Orange Flags (Serious Concern)

- 2+ reschedules
- Unprepared interviewers
- "We're like a family"
- Jokes about long hours
- Vague role description changes
- 10+ min late with no apology

### üî¥ Red Flags (Strong Exit Signal)

- Disrespectful treatment
- Inappropriate questions (age, family, etc.)
- Bait-and-switch on role/comp
- "Take it or leave it" pressure
- 3+ reschedules
- Ghosting for weeks
- Clear discrimination
